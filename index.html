I got into security the way a lot of people do: by breaking something I wasn’t supposed to, panicking a little, and then reading way too much about how it worked. These days I do it on purpose, with permission, and get paid for it.

My focus is on the bugs that live in application logic rather than in a CVE database — broken access control, authorization bypasses, authentication flaws, and the weird edge cases APIs leave wide open. I spend more time reading JavaScript bundles and API responses than running scanners.

That instinct doesn’t stop at the browser. I pop binaries open in Binary Ninja Ultimate when an API alone doesn’t tell the whole story, and I audit Solidity contracts for DeFi protocols through competitive contests on Code4rena. Different attack surface, same assumption: the logic is wrong until proven otherwise.

When I’m not chasing bounties, I’m writing up what I found (the parts I’m allowed to share), building small recon tools, and trying to get better at explaining impact clearly enough that triagers don’t have to ask twice.

Everything published on this site is for education. If you’re testing any of it yourself, make sure you’re explicitly authorized to.

focus areas

What I actually spend my time on

No two targets are the same, but most of my work falls into one of these buckets — web, binaries, or smart contracts.

Access Control & IDOR

Object-level authorization bugs, BOLA, and horizontal/vertical privilege escalation across web and mobile APIs.

Auth, Session & OAuth

Login flow abuse, JWT attacks, session fixation, and OAuth/SSO misconfigurations that lead to account takeover.

SSRF & Server-Side

Webhooks, URL previews, PDF/image processors, and anything that makes an outbound request on my behalf.

XSS & Client-Side

Reflected/stored/DOM XSS, CSP bypasses, and the fun edge cases introduced by modern frontend frameworks.

API & GraphQL Security

Introspection leaks, batching abuse, excessive data exposure, and missing field-level authorization.

Recon & Automation

Subdomain enumeration, JS analysis, and small pipelines that keep new attack surface flowing in automatically.

Reverse Engineering

Static and dynamic binary analysis in Binary Ninja Ultimate — unpacking, patch diffing, and figuring out what a binary actually does.

Smart Contract & DLT Security

Solidity/EVM logic bugs, reentrancy, access control, and economic exploits — audited through competitive contests on Code4rena.

Mobile & Native AppSec

iOS/Android app testing and native library review — insecure storage, hardcoded secrets, and the native code behind the UI.
